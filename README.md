# **Company Database**

This is a web application which management functionalities: add, delete, view, update information about employees. Application uses technologies: ReactJS client, Spring Boot server, MySQL database.

## Used technologies:
* Spring Web MVC
* Spring Boot
* JDBC
* ReactJS
* Bootstrap
* CSS
* HTML
* MySQL
* SQL
* IntelliJ Idea
* Visual Studio Code
